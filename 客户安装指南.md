# 📱 OpenClaw 安装指南（极简版）

---

## ⏱️ 预计耗时：20 分钟

---

## 步骤 1：安装 Termux（2 分钟）

1. 打开手机浏览器
2. 访问：https://f-droid.org/packages/com.termux/
3. 下载并安装 APK
4. 打开 Termux APP

---

## 步骤 2：授予权限（1 分钟）

看到黑色屏幕后，输入以下命令：

```bash
termux-setup-storage
```

手机弹出权限请求 → 点击"**允许**"

---

## 步骤 3：运行安装脚本（1 分钟）

复制以下命令，粘贴到 Termux 中：

```bash
curl -L https://raw.githubusercontent.com/mijunyang7/openclaw-mobile-installer/main/install.sh | bash

```

按**回车键**执行

---

## 步骤 4：等待安装（15-20 分钟）

脚本会自动运行，你会看到彩色文字输出：

```
[INFO] OpenClaw 一键安装脚本 v1.0.0
[OK] 基础依赖安装完成
[OK] OpenClaw 安装成功
...
```

⏳ **此时可以去做别的事，完成后会提示**

---

## 步骤 5：验证安装（1 分钟）

看到"安装完成"提示后，输入：

```bash
openclaw status
```

如果看到正常输出，说明**安装成功**！ ✅

---

## ✅ 完成！

现在 OpenClaw 已经安装好了，接下来：

1. 等待卖家远程配置
2. 或者查看使用手册自学

---

## ❓ 遇到问题？

### 问题 1：命令找不到

**错误提示**：`command not found`

**解决**：检查是否复制完整，确保命令以 `curl` 开头

---

### 问题 2：权限不足

**错误提示**：`Permission denied`

**解决**：
1. 手机设置 → 应用管理 → Termux → 权限
2. 确保"存储"权限已开启

---

### 问题 3：网络连接失败

**错误提示**：`Could not resolve host`

**解决**：
1. 检查 WiFi/流量是否正常
2. 换个网络再试
3. 联系卖家获取帮助
